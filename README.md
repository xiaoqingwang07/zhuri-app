# 逐日 — AI 目标拆解 + 朋友圈陪你坚持

> 你敢想，我敢拆——每天离目标近一点

**在线体验：** https://xiaoqingwang07.github.io/zhuri-app/

## 核心功能

### 🎯 AI 智能拆解
输入目标，AI 15秒内拆解为每日任务。支持100+种任务类型：
读书/运动/编程/考证/乐器/写作/艺术/语言学习/职场/备考/理财/冥想/饮食/育儿等。

不满意可以重新生成，确认后再开始执行。

### 🔥 连续打卡
每天打卡，连续徽章激励：
🌱 3天 → 🔥 7天 → ⭐ 14天 → 🏆 20天

### 💳 复活卡
漏了一天不怕——新目标送1张复活卡，可以补打卡。连续30天再奖励1张。

### 👥 社交监督
监督团里可以看到其他人的打卡状态。戳一下TA，TA会收到提醒。
（v0.3+ 支持真实后端同步）

### 📊 进度可视化
日历总览显示整体进度，完成率、近7天统计，以及"领先/落后"计划对比。

### 🌙 主题切换
浅色 / 深色 / 跟随系统，三档手动切换。

### 📤 数据导出
所有数据可导出为 JSON 备份，换设备不丢数据。

## 技术栈

- **前端：** Next.js 14 + TypeScript + Tailwind CSS
- **状态管理：** React useState + localStorage
- **AI：** Siliconflow API（DeepSeek-V3）+ Cloudflare Workers 代理
- **部署：** GitHub Pages（静态导出）

## 产品理念

逐日的核心问题是：**大目标太大，执行力跟不上，坚持不下去。**

解决方案：
1. AI 拆解 → 降低规划门槛
2. 每日打卡 → 制造损失厌恶
3. 复活卡 → 给了退路，不容易直接放弃
4. 社交监督 → 利用群体压力维持动力
